!Leçon DS 32: I-Adjectives
?x
#
# Fini par i (sauf ie)
# Connecte avec un i l'adjectif et le nom
# Vire le i en fin et :
# -i          : présent
# -kunai      : négation
# -katta      : passé
# -kunakatta  : négation passée
# Possibilité de finir une phrase par l'adjectif qui remplace desu
# On n'utilise jamais da à la place de desu avec les adjectifs en i

!Leçon DS 34: Emotions
?x
#
# Si le sujet est évident, il est homis.
# Les sujets les plus souvent homis sont : je, tu, il
# Pour les émotions, on parle souvent de soi -> pas de watashi wa.
#
# Verbe :
# - conjugué en base te + iru : présent, le sujet est dans cet état
# - conjugué en base ta : passé, le sujet a fini dans cet état
#
# Pour utiliser "amour" (ai) comme une émotion (c'est un nom à la base),
# nécessité de mettre suru après et d'utiliser un nom.
# ex : Je t'aime -> anata wo ai shite iru
